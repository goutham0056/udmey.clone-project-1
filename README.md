# udmey.clone-project-1
Udemy Clone Website

This project is a front-end clone of the Udemy website created using HTML and CSS. The goal of this project is to practice web development skills by replicating the layout and design of a popular online learning platform.

The website includes a responsive layout with important sections similar to the original Udemy website.

Features

- Navigation bar with logo, search bar, and menu
- Course category buttons
- Recommended courses section
- Topics recommended for you
- Most popular courses section
- Footer with useful links and copyright information

Technologies Used

- HTML5
- CSS3
- Flexbox for layout design

Purpose

This project was built as part of my learning journey in front-end development to improve my skills in HTML structure, CSS styling, layout design, and responsive web design.

Project Status

Completed as a practice project to demonstrate basic front-end development skills.
